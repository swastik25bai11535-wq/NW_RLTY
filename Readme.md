# Reminder and To-Do List App

## Overview
This project is a console-based reminder and to-do list application developed in Python. It allows users to securely log in, receive timely reminders, and manage their daily tasks by marking them completed and tracking progress. The app aims to improve productivity and task management for students and professionals.

## Features
- Password-based authentication system to secure access.
- Time-based reminders for morning, afternoon, and evening.
- To-do list management with task addition, completion tracking, and status display.
- Congratulatory feedback when all tasks are completed.

## Technologies and Libraries Used
- Python 3.x
- `datetime` module for time-based reminders
- Object-oriented programming with classes for modular design

## Usage
1. When you run the program, it will prompt for a password. Enter the preset password to gain access.
2. Upon successful login, reminders for current time slots (morning, afternoon, evening) will be displayed.
3. The to-do list management will run, showing completed and remaining tasks as they are updated.
4. Follow on-screen prompts and review task status updates.
